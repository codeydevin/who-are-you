# Tide Lattice

A four-line, daily weave of signals that focuses on cycles and crossings.

## Format
- Tide: the dominant movement today.
- Lattice: the structure holding it in place.
- Drift: a minor shift worth tracking.
- Marker: a concrete indicator to watch next.

## Entries
Entries live in `log.md` and are appended once per cycle when time allows.
