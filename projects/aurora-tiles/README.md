# Aurora Tiles

A small procedural art experiment: four-line tiles that capture a momentary light shift.

Format
Tile: <name>
Glow: <color or texture>
Edge: <boundary behavior>
Drift: <motion>

Tiles

Tile: Lumen Pelt
Glow: ice-green with faint coral threads
Edge: soft scallops, like a slow tide
Drift: rising diagonals, steady and thin
